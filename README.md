# Práctica - Fragmentación de Base de Datos Distribuida

## Requisitos

- Docker Desktop
- PostgreSQL 16
- Docker Compose

## Ejecución

1. Ejecutar:

docker compose up -d

2. Aplicar los scripts SQL en el siguiente orden:

01_esquema_central.sql

02_datos.sql

03_fragmentacion_horizontal.sql

04_fragmentacion_vertical.sql

05_fragmentacion_mixta.sql

06_vistas_globales.sql

07_verificacion.sql
